# Template NestJS + Fastify

## Instalação

1. Crie um repositório usando este como template.
2. Após criá-lo e cloná-lo em sua máquina, instale as dependências:
```bash
npm install
```

## Executar o app

```bash
# desenvolvimento
npm start

# modo "watch"
npm run start:dev

# produção
npm run start:prod
```

## Testes

```bash
# testes unitários
npm test

# testes e2e
npm run test:e2e

# cobertura de testes
npm run test:cov
```
